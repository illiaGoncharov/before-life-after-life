Интерактивный арт-проект, исследующий жизненный цикл через призму личных архивов и искусственного интеллекта.

## О проекте

Before Life | After Life — это веб-приложение, представляющее собой интерактивный арт-проект. Проект исследует, как личные архивы могут сохранить нечто большее, чем индивидуальный опыт, и как они могут интерпретировать современный мир через призму архивных изображений.

Приложение позволяет пользователям взаимодействовать с визуальным контентом через различные режимы просмотра и загружать собственные материалы, которые затем трансформируются с помощью искусственного интеллекта.

## Технологии

Проект разработан с использованием современных веб-технологий:

- **Frontend**: React 19
- **Стилизация**: CSS Modules
- **Оптимизация**: 
  - Lazy Loading компонентов
  - WebP формат изображений
  - Code splitting
- **Типографика**: Кастомные шрифты Hershey-Noailles
- **Хранение данных**: Firebase Storage (изображения), Firestore (данные формы)
- **Производительность**: 
  - First Contentful Paint < 2s
  - 60fps анимации
  - Touch response < 100ms

## Структура проекта

```
/src
├── /assets
│   └── /fonts               # Кастомные шрифты Hershey-Noailles
├── /components
│   ├── /App                 # Корневой компонент
```

## Функциональность

Приложение включает несколько режимов взаимодействия:

- **Gallery**: Коллекция изображений, исчезающих при наведении
- **Text**: Прокручивающийся текст с аудио-сопровождением
- **Before Life**: Форма для загрузки пользовательских изображений
- **After Life**: Информация о проекте и его создателях

### Мобильная версия

Мобильная версия представляет собой упрощенную структуру:

1. **Центральная галерея** - основной элемент без сложной логики
2. **Before Life** - адаптированная форма для мобильных устройств
3. **After Life** - адаптированный лендинг с описанием проекта

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/before-life-after-life.git
cd before-life-after-life

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm start

# Сборка для продакшена
npm run build

# Деплой на GitHub Pages
npm run deploy
```

## Деплой

Проект настроен на автоматический деплой через GitHub Actions. При каждом пуше в ветку `main` сайт автоматически собирается и деплоится.

**Важно:** Перед первым деплоем нужно:
1. Включить GitHub Pages в настройках репозитория (`Settings` → `Pages` → `Source: GitHub Actions`)
2. (Опционально) Настроить секреты для EmailJS в `Settings` → `Secrets and variables` → `Actions`

Подробная инструкция: [docs/DEPLOY.md](./docs/DEPLOY.md)
