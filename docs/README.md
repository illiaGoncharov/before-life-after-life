## Структура проекта

```
/src
├── /assets
│   └── /fonts               # Кастомные шрифты Hershey-Noailles
├── /components
│   ├── /App                 # Корневой компонент
│   ├── /Form                # Форма "Before Life"
│   ├── /Gallery             # Галерея изображений
│   ├── /Text                # Текстовый режим
│   ├── /ByPrompt            # Фильтр по промптам
│   └── /ByContributor       # Фильтр по авторам
├── /data
│   ├── prompts.js           # 45 промптов жизненного цикла
│   └── contributors.js      # 5 авторов/контрибьюторов
└── /ui
    ├── /header              # Компонент хедера
    └── /footer              # Компонент футера
```

## Критерии готовности

1. **Функциональность**
   - Все компоненты работают без ошибок
   - Сохранена существующая логика
   - Реализована мобильная версия

2. **Производительность**
   - First Contentful Paint < 2s
   - 60fps анимации
   - Оптимизированные изображения

3. **Доступность**
   - Корректная работа на всех целевых устройствах
   - Поддержка клавиатурной навигации
   - ARIA-атрибуты для улучшения доступности
